# Silbentrenner

Eine kleine Web-App, mit der du Ballermann-Songzeilen per Knopfdruck über die ChatGPT API in
Silben trennen kannst. Ideal, um die Silbenanzahl pro Zeile zu vergleichen.

## Nutzung

1. Öffne `index.html` lokal oder über GitHub Pages.
2. Füge deinen ChatGPT API-Key ein.
3. Tippe deinen Songtext ein (eine Zeile pro Zeile).
4. Drücke pro Zeile auf „+“, um die Silbentrennung rechts zu sehen.

> Hinweis: Der API-Key wird im Browser gespeichert (LocalStorage). In öffentlichen Deployments
> solltest du deinen eigenen Key verwenden und ihn nicht teilen.
