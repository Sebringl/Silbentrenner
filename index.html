Ballermann-Tool

Silbentrenner für Songzeilen

Tippe deinen Songtext ein, klick pro Zeile auf „+“ und erhalte die Silbentrennung rechts daneben. Die Trennung erfolgt über die ChatGPT API.

Der Key bleibt nur in deinem Browser (LocalStorage) gespeichert. Für GitHub Pages solltest du keinen Key teilen, den andere mitnutzen.

Tipps für gleichmäßige Silben

  • Verwende kurze Wörter und wiederkehrende Muster für den Refrain.
  • Halte die Silbenanzahl pro Zeile ähnlich, damit der Groove stabil bleibt.
  • Nutze den „+“-Button, wenn du eine Zeile fertig hast.
